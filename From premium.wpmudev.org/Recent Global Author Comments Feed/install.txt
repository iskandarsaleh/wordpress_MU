Plugin Name: Recent Global Author Comments Feed
Author: Andrew Billits

Files:
----------------------------------------------------------------------

recent-global-author-comments-feed.php - goes in /wp-content

Instructions:
----------------------------------------------------------------------
1) Place the files in the specified directories

Feed is located at:
http://yourdomain.tld/wp-content/recent-global-comments-feed.php?author=USERID

Replace USERID with the user id of the author.

Please note that this widget requires the Comment Indexer plugin to be installed:
http://premium.wpmudev.org/project/comment-indexer
